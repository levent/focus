# Focus: daily prompt

This app is intended to focus thinking and productivity day to day. 

Every day I want you to prompt the user based on these conditions:
- check if there’s a focus entry for today
- If there is not, then prompt them
    - “What is your primary focus for today?” (Wait for an answer)
    - “What blockers do you envisage?” (Wait for an answer)
    - Persist those (schema will follow)
- If there are entries for today, prompt the user with
    - Did you manage to achieve your focus goal? (Wait for an answer)
    - If they answered in the negative, ask for a reason (wait for an answer)
    - Persist those (schema will follow)
